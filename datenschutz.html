Externe Links & Drittanbieter

Diese Website enthält Links zu externen Diensten. Beim Aufruf solcher Links oder beim Laden externer Ressourcen (z. B. Bannerbilder) können Daten an die Betreiber dieser Dienste übertragen werden (z. B. Referrer, IP-Adresse, ggf. gesetzte Cookies).

  • Discord: Wir verlinken zur Discord‑Community und verwenden serverseitig die öffentliche Invite‑API, um grobe Statistiken (approximate_member_count, approximate_presence_count) abzurufen. Dabei werden keine personenbezogenen Daten Ihrer Besucher von uns mit Discord geteilt; die API‑Abfrage erfolgt serverseitig durch GitHub Actions.
  • Instagram / Meta: Beim Aufruf können Daten an Instagram/Meta gesendet werden; wir binden keine Instagram‑Plugin direkt ein, verlinken jedoch auf das Profil.
  • Gamertransfer: Wir verlinken auf Gamertransfer und laden ein Bannerbild von deren Servern.
  • Deutsche eSport Bundesliga (DeSBL): Externer Link zum Teamprofil.
  • GitHub / GitHub Actions: Die Seite wird über GitHub Pages gehostet. Unser Repository verwendet eine GitHub Action, die periodisch die öffentliche Discord‑Invite‑API abfragt und eine statische JSON‑Datei mit den aktuellen Zahlen erzeugt. Diese Aktion verwendet ausschließlich öffentliche API‑Daten und Ihr Besuch der Website ist davon nicht betroffen.

Bitte beachten Sie die Datenschutzhinweise der jeweiligen Drittanbieter (z. B. Discord, Instagram/Meta, Gamertransfer). Wir empfehlen, diese vor dem Besuch der verlinkten Seiten zu lesen, wenn Sie Bedenken hinsichtlich der Datenverarbeitung haben.
